## Installation

### Dependencias
- Makefile
- Docker

The full list of common commands used in various situations:
* ``make help``

#### Step 1: Build
Build a new Docker image:
```
make build
```
#### Step 2: Compile
Compile the source code:
```
make compile
```
Resulting artifacts can be found in the `bin` directory.
#### Step 3: Run
Run the compiled source code:
```
make run
```
